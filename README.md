## Résumé
Dans ce projet nous avons effectué la modélisation d'une base de données dédiée à la gestion 
complète des compétitions sportives, incluant les équipes, les joueurs, les entraîneurs et les matchs. 
Dans ce projet nous avons définit les relations structurelles essentielles pour organiser les tournois
et les scores, de même nous avons proposr des variantes supplémentaire pour pouvoir englober toute les competitions.
