## Résumé
  Dans ce projet nous avons effectué la modélisation d'une base de données dédiée à la gestion 
  complète des compétitions sportives, incluant les équipes, les joueurs, les entraîneurs et les matchs. 
  Aussi nous avons définit les relations structurelles essentielles pour organiser les tournois, les scores
  et nous avons proposr des variantes supplémentaire pour pouvoir englober toute les competitions.
